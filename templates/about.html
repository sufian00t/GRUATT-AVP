Welcome to GRUATT-AVP Web Server

To support the efficient identification of anti-viral peptides and provide broad access to the scientific community, we have designed and implemented the GRUATT-AVP web server.

How to Use the GRUATT-AVP Web Server:
  1. Access the Web Server:
    Open the web server here at https://gruatt-avp.vercel.app/
  2. Input Your Data:
    Enter your query in FASTA format into the provided text box.
  3. Submit for Prediction:
    Click on the “Submit” button to initiate the prediction process. The server typically processes the data within a few seconds.
  4. View Results:
    Once the prediction is complete, results will be displayed directly on the web server interface.
